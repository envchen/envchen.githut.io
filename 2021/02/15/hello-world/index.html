Flag-raising-platform

前后端分离线上升旗平台

后端工作区 flag

前端工作区 flag-vue（待创建）

后端区

所用技术:

技术 版本 描述
Java JDK15 编程语言
SpringBoot 2.4.2 后端框架
Maven 3.6.3 项目构建工具
MyBatis - 持久层框架
MySQL 8.0 关系型数据库
Redis - Key-Value数据库
Lombok - 代码简化工具
Screw 1.0.3 数据库文档生成工具

数据库设计：

数据库名：flag

数据表名：stu

数据列：

序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 yb_id varchar 16 0 N N 易班id
2 yb_realname varchar 16 0 N N 姓名
3 yb_studentid varchar 16 0 N Y 学号
4 yb_classname varchar 16 0 N N 班级
5 yb_collegename varchar 16 0 N N 学院

数据表名：msg

数据列：

序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 yb_studentid varchar 16 0 N N 学号
2 rank int 10 0 N Y 升旗排位
3 date datetime 19 0 N N 升旗时间
4 accessToken varchar 64 0 N N 所用授权码

测试接口：

1.地址：http://47.115.84.57:8221/api/211/user/raise

方式：POST

参数：accessToken

描述：普通用户升旗用

2.地址：http://47.115.84.57:8221/api/211/admin/allUser

方式：POST

参数：accessToken

描述：管理员用，获取所有升旗记录
